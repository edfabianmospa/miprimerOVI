¿QUE ES SOFTWARE?

Son los programas que tienen cada sistema operativo para la ejecutacion de diversas herraminetas informativas, juegos etc.

Existe software libre y Licenciado: el concepto de software libre se utiliza para decir que los programas son totalmente compartidos, que se pueden copiar y modificar.

A lo contrario del software licenciado el cual tiene restricciones por derechos de autor el cual no te permiten la distribución y modificación.

También existe software educativo el cual son programas totalmente enfocados en informática, creados para diseñar y facilitar al estudiante a su proceso de aprendizaje.

Para terminar hay que tener en cuenta que el software es interrelacionar con respecto al hardware el cual es total mente lo contrario siendo el hardware toda la parte física del computador.
